# lcdBrightness2
GY-30 BH1750FVI 光センサー で明るさを取得して Raspberry Pi 公式7インチタッチスクリーンの輝度を調整するプログラム。（オブジェクト指向版）  

■使い方  
Python で lcdBrightness2 を起動する。  

/usr/bin/python /root/lcdBrightness2/main.py
